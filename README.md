# EATS

### Commands

| Group  | Command  | Operater           |
| ------ | -------- | ------------------ |
| Basic  | keywords | `what's happening` |
| &nbsp; | exact    | `"happy hour"`     |
| &nbsp; | or       | `(cats OR dogs)`   |
| &nbsp; | minus    | `-cats -dogs`      |
| &nbsp; | tag      | `#bigdog`          |

### Usage

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
